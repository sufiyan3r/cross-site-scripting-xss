Demonstration of Reflected XSS Vulnerability

To demonstrate the Reflected XSS vulnerability, you can try the following steps:

  1. The current web page comprises of 2 forms. One is protected and the other is vulnerable.
  2. Copy & enter the following code into the 'full name' input box below [5]:
    3. <script>alert("Hello world!");</script>
  3. Submit the form and observe the results

Expectation of Results

If the web application is 'protected' from Reflected XSS, the alert message will not be executed and instead it will prompt the user with a message on top of the screen, protecting the user to security risks.

If the web application is 'vulnerable' to Reflected XSS, the alert message will be executed when the search results page is loaded, potentially exposing the user to security risks.
